Instructions on how to create a webpage on the production server.

  1. Create an index.html file
  2. Do whatever you need to do to edit the webpage, and then validate the file.
  3. Once it has no errors, save it.
  4. Go into sftp from Terminal by typing "sftp student1234@cmis-hsteacher"
  5. Put the file onto the sftp server by typing "put index.html"
  6. Go to your browser and type "http://cmis-hsteacher/~student1234/" into the adress bar to view your webpage
